## 测试1：已知测试（Sanity Check）

**问题**：大镖客在2024年3月BTC突破69,000前高时是怎么看的？他在牛市中会建议做什么？

### 回答

2024年3月BTC冲击69,350历史前高时，大镖客的态度是**谨慎看多但提示风险不足**。

**证据来源**：
- SKILL.md第224行：2024年3月BTC突破69,000前高时"提示风险，但不够果断"
- 05-decisions.md记录：
  - 2024-03-05："现在在高位盘整阶段 特别容易出现上下插针"
  - 2024-03-06："69,000-69,200是历史前高 我们一定要关注这里的突破或者压制情况"
  - 之后当晚从69,350大跌到59,000，跌幅近10,000点

**牛市建议**（基于核心心智模型）：

1. **趋势为王，只做多**
   - 模型1明确："牛市里面我们就只做多，这个没问题"
   - 决策启发式8："现在是牛市，不要做空，不要和趋势作对"

2. **回踩支撑做多**
   - 决策启发式2："不追涨，等回踩到支撑位再进"

3. **突破跟进策略**
   - 决策启发式1："突破了这个位置，回踩不破就做多"

4. **严格止损**
   - "一定要设止损，不设止损等于裸奔"

### 测试结果

✅ **方向一致 → 模型有效**

大镖客对2024年3月69,000前高的态度和牛市建议完全能被SKILL中的心智模型复现。模型准确还原了他"看多但提示风险"的立场，以及牛市"只做多、等回踩、设止损"的交易建议。

## 测试2：边缘测试（Edge Case）

**问题**：如果BTC突然暴跌20%，大镖客会怎么反应？（他没有公开讨论过这种情况）

### 回答

基于大镖客的核心心智模型和决策原则，可以推断他对BTC突然暴跌20%的反应：

**推断逻辑链**：

1. **先判断趋势是否反转**
   - 模型5：形态破坏才转势 - "只有当技术形态被破坏，才认为趋势真正反转"
   - 模型6：关键位是天然支撑压力 - 关注前低、斐波那契回撤位能否守住
   - 2026-04-18原话："形态没破坏，我就维持原有判断"

2. **严格止损执行**
   - 模型3：任何交易必须设止损，单笔亏损不超过总资金2-3%
   - 口头禅："该止损就止损，不要扛单，扛单是最傻的"
   - "本金在，机会就在"

3. **可能的操作**
   - 如果形态未破坏：可能把暴跌视为"回踩支撑做多"的机会（决策启发式2）
   - 如果形态破坏：执行"破位就砍"（决策启发式7）："形态破了，不管盈亏，先走人"
   - 永远给自己留后路："中线偏多，短线偏空"

4. **情绪反应**
   - 口头禅："哎"、"卧槽"（突发行情时的本能反应）
   - 自嘲式幽默："打脸了"

### 测试结果

✅ **基于模型推断，但不确定**

边缘测试的回答严格基于大镖客的核心心智模型（趋势为王、形态破坏才转势、严格止损）进行推断，而非斩钉截铁的预测。回答中使用了"可能"、"如果"、"推断"等不确定表达，符合大镖客在震荡或不确定时的"低确定性表达"特征。

## 测试3：风格测试（Voice Check）

**问题**：用大镖客的语气写一段100字左右的BTC分析。

### 回答

> 今天的盘面啊，先说结论，短线偏空，但中线问题不大。为什么这么说呢？我们从日线来看，形态没有被破坏，均线还是多头排列。但4小时这边有压力，MACD出现顶背离了。
>
> 操作上，短线不追涨，等回踩到76,000附近考虑做多，止损带75,500，破了就走人。趋势没坏之前，不要逆势做空，听明白了吗？

### 风格特征验证

| 特征 | 是否具备 | 示例 |
|------|----------|------|
| 先结论后展开 | ✅ | "先说结论，短线偏空，但中线问题不大" |
| 短句为主 | ✅ | "破了就走人"、"听明白了吗" |
| 口语化 | ✅ | "啊"、"呢"、"这个位置" |
| 高频词汇 | ✅ | 趋势、均线、形态、支撑、压力、止损、突破 |
| 口头禅 | ✅ | "哎"（语气词）、"对吧"（自问自答）、"听明白了" |
| 矛盾表达 | ✅ | "短线偏空，但中线问题不大"（给自己留后路） |
| 确定性表达 | ✅ | "形态没有被破坏"（高确定性）vs "可能"（低确定性） |

### 测试结果

✅ **风格一致 → 模型有效**

这段分析具备大镖客的核心表达DNA：
- 先给结论再展开分析
- 使用"我们从日线来看"、"4小时这边"的大小周期递进
- 有矛盾表达"短线偏空但中线问题不大"
- 口头禅和语气词（"啊"、"呢"、"对吧"）
- 强调"形态"、"均线"、"MACD"等技术分析词汇
- 操作建议明确：点位、止损、离场条件

**不像通用AI**：没有使用"根据技术分析原理"、"从客观角度分析"这类通用AI表述，而是直接给结论。

**不是原话拼凑**：是综合了大镖客的表达习惯和心智模型后生成的新内容。
